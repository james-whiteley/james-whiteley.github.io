ABOUT ME

I’ve been working with Javascript, HTML and CSS since all the way back in 2000, when I was 12. Back then I decided to get into programming because I wanted to build things that people would want to use (... and after a friend sold a website for £500 - £847.16 inflation adjusted - I figured I was going to be rich). I started hacking Javascript games, tweaking and experimenting and I haven’t stopped programming like it since.

My interest was piqued after them early days and I decided to study Computer Science at the University of Liverpool, developing my skills in full stack development. Since then there’s been a snowball effect, picking up skills in various programming languages and technologies as I go.
